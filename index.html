/* An interesting behavier of display: grid and place-items: center is that if there is another element (such as this paragraph), it won't be able to center the intended specific element. So, display: grid is just an opcion if there is just one element. However, it is possible to use position: absolute in the extra elements to make them stopping interfering in the behavior of the intended element one wants to center */
